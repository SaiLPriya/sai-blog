## ğŸ’¡ Approach 1: Using a Set (Simple and Clear)

We can use a **Set** to keep track of seen numbers.

1. Create an empty set `seen`.
2. Create another set or array `duplicates`.
3. For each number:

   * If it's not in `seen`, add it.
   * If itâ€™s already in `seen`, add it to `duplicates`.

---

### âœ… JavaScript Code

```javascript
function findDuplicates(nums) {
  const seen = new Set();
  const duplicates = new Set();

  for (let num of nums) {
    if (seen.has(num)) {
      duplicates.add(num);
    } else {
      seen.add(num);
    }
  }

  return Array.from(duplicates);
}

// Example:
console.log(findDuplicates([4, 3, 2, 7, 8, 2, 3, 1])); // [2, 3]
```

```javascript
function findDuplicates(nums) {
  const result = [];

  for (let i = 0; i < nums.length; i++) {
    let index = Math.abs(nums[i]) - 1;
    if (nums[index] < 0) {
      result.push(Math.abs(nums[i]));
    } else {
      nums[index] = -nums[index];
    }
  }

  return result;
}

// Example:
console.log(findDuplicates([4, 3, 2, 7, 8, 2, 3, 1])); // [2, 3]
```
